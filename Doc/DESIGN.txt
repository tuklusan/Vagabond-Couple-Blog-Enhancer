--------------------------------------------------------------------------------
1. PURPOSE
--------------------------------------------------------------------------------
Download existing Blogger posts by title, rewrite their prose using NVIDIA NIM
(Llama), inject a TravelAction ld+json schema block, and save both the original
and rewritten HTML locally for manual review.

No automatic publishing. The user decides if and when to upload.

--------------------------------------------------------------------------------
3. PIPELINE
--------------------------------------------------------------------------------
Step  Script                    Description
----  ------------------------  -------------------------------------------------
  1   step1_fetch.py            Authenticate with Blogger API (read-only OAuth2)
                                Search for post by title, fetch full HTML content
                                Save original HTML to Input/
  2   step2_extract.py          Parse HTML with BeautifulSoup
                                Extract text from p/h1-h6/li/blockquote tags
                                Save numbered prose blocks to Temp/prose_blocks.txt
  3   step3_rewrite.py          Send prose blocks to NVIDIA NIM (Llama 3.1 70B)
                                Save rewritten blocks to Temp/rewritten_blocks.txt
  4   step4_reinsert.py         Parse original HTML, replace prose text nodes
                                with rewritten text. Save to Temp/rewritten_html.html
  5   step5_inject_ldjson.py    Build TravelAction ld+json schema block
                                Insert immediately before <!--more--> jump break
                                Save to Temp/final_html.html
  6   step6_save_output.py      Copy Input original (already saved in step 1)
                                Write final HTML to Output/<title>__<timestamp>.html
  7   step7_review.py           Send final HTML to DeepSeek for quality review
                                Save review to Temp/output_review.txt
                                Exit code 1 if Critical issues found (non-blocking)
  8   step8_quality_gate.py     Writer+auditor feedback loop (up to 3 rounds)
                                Writer: Mistral Large 3 primary / Mistral Medium 3.5 fallback (NIM)
                                Auditor: DeepSeek V4 Pro via NVIDIA NIM
                                Checks: writing rules, travelogue sanity, route/chronology,
                                POI accuracy, natural language flow
                                Overwrites Temp/final_html.html + latest Output/ file if revised
                                Saves Temp/quality_gate_report.txt

--------------------------------------------------------------------------------
4. SCOPE CONSTRAINTS
--------------------------------------------------------------------------------
- No uploads to Blogger under any circumstances — output is local only
- OAuth scope: blogger.readonly
- Writing rules default to Config/writing_rules.md
- All intermediate files go to Temp/ — never to AppData or system temp
- All commands run in the foreground — never in background processes
